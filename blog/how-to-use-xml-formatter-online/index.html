Common Mistakes to Avoid

Honestly, I've made all of these. The most common one? Forgetting that XML is case-sensitive. If you have <User> and </user>, the formatter will throw an error. It isn't broken; your code is.

Another big one is the "Prolog" error. That's the <?xml version="1.0"?> tag at the very top. If you accidentally paste some random text before that tag, the xml formatter might get confused. Keep it clean. Start with the bracket.
