Pro Tip: Mind the Size

Never print a QR code smaller than 2cm x 2cm (0.8 inches). Anything smaller and the physical sensor on a smartphone camera literally can't resolve the individual squares. If you have a lot of data, the code gets denser, meaning you need to print it even larger.

And honestly? Use the SVG format if you're doing anything for print. If you use a PNG and try to blow it up for a window sign, it'll look like a pixelated mess from 1995. SVG is "infinitely scalable," meaning it stays crisp whether it's on a postage stamp or a billboard.
